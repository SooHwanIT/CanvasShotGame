## 시작 화면 
![image](https://user-images.githubusercontent.com/32871806/124858914-e6f32300-dfe9-11eb-816a-af19be22ebff.png)
### 화면을 클릭시 해당 방향으로 총알을 날릴 수 있음
![image](https://user-images.githubusercontent.com/32871806/124858972-fe321080-dfe9-11eb-927e-df98bc9477e0.png)
### 랜덤한 크기의 적들이 화면 테두리 쪽에서 생성되어 중앙으로 옴
![image](https://user-images.githubusercontent.com/32871806/124859266-8fa18280-dfea-11eb-8e12-25e51e98b0df.png)
### 총을 맞은 적들은 파편을 튀기며 크기가 천천히 작아지며 일정 이하로 작아진 적들은 제거됨
### 점수(=score)은 적을 맞췄을때 얻을수 있으며 적이 제거되었을때 추가적인 점수를 얻음
![image](https://user-images.githubusercontent.com/32871806/124859481-00e13580-dfeb-11eb-976e-d6a8c54abba5.png)
### 시간이 지날수록(=점수가 오를수록) 화면이 붉어지고 적들이 빨라짐(잔상의 길이로 속도를 확인할 수 있음)
![image](https://user-images.githubusercontent.com/32871806/124859239-80bad000-dfea-11eb-972c-3f82266934d5.png)
### 적이 플레이어에게 닿으면 총 점수가 출력됨, 버튼을 눌러 다시 시작할수 있음
